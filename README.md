# Bharath Karumanchi – Portfolio

Live at: **https://bharath021.github.io/bharath-portfolio**

Built with React + Vite. Features 3D isometric Canvas animations for every project.

## Local Development

```bash
npm install
npm run dev
```

Open http://localhost:5173

## Deploy to GitHub Pages

This repo auto-deploys via GitHub Actions on every push to `main`.

Just push your code and visit your live URL in ~2 minutes.
